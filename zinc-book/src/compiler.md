# Compiler CLI

## Interface

```
znc 0.5.0
The Zinc compiler

USAGE:
    znc [FLAGS] --input <input> --output <output>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -i, --input <input>      Specifies the *.zn input file name
    -o, --output <output>    Specifies the *.znb output file name
```
