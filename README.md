# The ZRust compiler

## Roadmap

### Version 0.1

- [x] primitive types 
- [x] input and witness (exported as JSON output)
- [x] var declarations
- [x] operators on primitive types
- [x] `require()`
- [x] comments
- [x] `debug!()`
- [x] interpreter

### Version 0.2

- [x] syntax highlighting for Visual Studio Code
- [x] mutability
- [x] conditionals
- [x] `for` loops (without `while`)

### Version 0.3

- [x] standard library (from bellman)

### Version 0.4

- [x] `for` loops with `while`
- [x] arrays
- [x] tuples
- [x] type aliases
- [x] structures
- [x] code generation of all of the above

### Version 0.5

- [x] functions
- [x] modules and imports
- [x] C-like `enum`
- [x] `match`

### Version 0.6

- [ ] conditional optimization
- [ ] std lib
- [ ] `Option<>`

### Version 0.7

- [ ] `Result<>`, ? operator
- [ ] simple references (more like aliases)
- [ ] move()/copy()

### Later

- [ ] `unsafe_unchecked`
- [ ] testing framework with coverage metrics
- [ ] bytes and strings
- [ ] interfaces?
- [ ] `unsafe_rust`?
- [ ] formal verification
